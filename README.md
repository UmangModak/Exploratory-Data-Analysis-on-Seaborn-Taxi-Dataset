# Exploratory-Data-Analysis-on-Seaborn-Taxi-Dataset
This repository contains a detailed Exploratory Data Analysis (EDA) of the Seaborn Taxi dataset, providing insights into taxi trip patterns, temporal trends, and data quality considerations. The analysis highlights key patterns in taxi demand, trip duration, fare, and payment behavior.

| Column          | Description                       |
| --------------- | --------------------------------- |
| pickup          | Pickup timestamp of the trip      |
| dropoff         | Dropoff timestamp of the trip     |
| passengers      | Number of passengers              |
| distance        | Distance of trip (in miles/km)    |
| fare            | Fare charged                      |
| tip             | Tip amount                        |
| tolls           | Tolls paid                        |
| total           | Total amount (fare + tip + tolls) |
| color           | Taxi color (Yellow/Green)         |
| payment         | Payment mode (Credit Card/Cash)   |
| pickup_zone     | Pickup location zone              |
| dropoff_zone    | Dropoff location zone             |
| pickup_borough  | Borough of pickup                 |
| dropoff_borough | Borough of dropoff                |
